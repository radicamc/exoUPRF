Installation
============

The latest release of **exoUPRF** can be downloaded from PyPI by running:

.. code-block:: bash

    pip install exouprf

Alternatively, the package can be grabbed from GitHub via:

.. code-block:: bash

    git clone https://github.com/radicamc/exoUPRF.git
    cd exoUPRF
    python setup.py install
